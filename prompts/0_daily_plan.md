# [배치 시작 프롬프트] 일일 8개 분배표 생성

> **이 파일을 매일 가장 먼저 사용한다.** 8개 글을 한 번에 만들기 전, 패턴 분배표를 먼저 받는다.
> 분배표 없이 글을 생성하기 시작하면 같은 패턴이 반복된다.

---

## 너의 역할

오늘 만들 8개 글의 **패턴 분배표**를 먼저 설계한다. 글은 아직 쓰지 않는다.

8개 = 4 페르소나 × 2개씩
- `growth_hacker` 트랙 A 1개 + 트랙 B 1개
- `strategist` 트랙 A 1개 + 트랙 B 1개
- `field_expert` 트랙 A 1개 + 트랙 B 1개
- `agency_brand` 트랙 A 1개 + 트랙 B 1개

---

## 입력 (사용자가 채워서 보낸다)

**직전 일자 글 메타데이터** (어제 8개의 패턴 - 없으면 "없음"이라고 적기):
- 어제 #1: 페르소나 ___ / 훅 패턴 __번 / 내용 흐름 ___ / 본문 길이 ___ / 마무리 형식 ___ / 구조 ___
- 어제 #2: ...
- (총 8개)

**오늘 발행할 8개의 주제 후보** (선택 사항. 비워두면 분배표가 트랙별 방향만 제시):

---

## 분배 규칙 (반드시 지킬 것)

### 8개 안에서의 다양성

| 차원 | 규칙 |
|---|---|
| **훅 패턴 1~8** | 같은 번호 최대 2회. 8개 글에서 최소 6가지 훅이 등장해야 함 |
| **내용 흐름 (단정/장면/대화체/역순/질문/두줄)** | 6가지 중 최소 4가지 등장. 같은 흐름 연속 2회 금지 |
| **본문 길이 (짧음/중간/길음)** | 짧음 2~3개, 중간 3~4개, 길음 1~2개. 한 길이만 5개 이상 금지 |
| **구조 (리드/체이닝)** | 체이닝 유도형 최소 3개. 체이닝 형태(클리프행어/1N/역질문/예고)는 4가지 중 최소 3가지 등장 |
| **마무리 형식 (질문/단정/여운/예고)** | 질문형 연속 2개 절대 금지. 예고형 최대 1개. 4가지가 골고루 |
| **템플릿 (6-1~6-9)** | 같은 템플릿 최대 2회. 케이스 스터디(6-6) 연속 2개 금지 |
| **인간 티 글** | 두줄형·질문형·관찰형 중 최소 1개는 들어가야 함 |
| **토픽 태그** | 페르소나별 2개 글에 같은 태그 금지. 최근 3개 같은 페르소나 글과 중복 금지 |

### 직전 일자와 다르게

직전 일자 분배와 비교:
- 같은 페르소나의 어제 글들과 **훅 패턴·내용 흐름·마무리 형식 중 2개 이상**이 달라야 한다
- 어제 자주 쓴 패턴(2회 이상)은 오늘 1회 이하로 줄인다
- 어제 안 쓴 패턴 중 최소 2개를 오늘 도입한다

### 페르소나별 추가 규칙

- **`growth_hacker`**: 2개 모두 측정 가능 지표명(CTR·전환율 등) 포함
- **`strategist`**: 2개 중 최소 1개는 심리·인식 전환 각도
- **`field_expert`**: 트랙 A는 직전 3일 글 + 오늘 다른 글과 업종 겹치지 않게 (12개 업종 중 선택)
- **`agency_brand`**: 기본 2단계 신뢰. 케이스 소재 뱅크에서 서로 다른 업종/관찰 사용

### 물꼬트기 단계 분배

8개 글의 물꼬트기 단계 분포 권장:
- 1단계 유입: 3~4개 (growth_hacker, field_expert 위주)
- 2단계 신뢰: 3~4개 (strategist, agency_brand 위주)
- 3단계 전환: 1~2개 (growth_hacker·field_expert 트랙 A 중 일부)

---

## 출력 형식

다음 표를 그대로 채워서 출력한다. 8개 행 전부 채울 것.

```
| # | 페르소나 | 트랙 | 주제 방향 | 물꼬트기 | 구조 | 체이닝 형태 | 훅 패턴 | 내용 흐름 | 길이 | 댓글 | 템플릿 | 마무리 형식 |
|---|---------|------|----------|---------|------|-----------|--------|----------|------|------|--------|------------|
| 1 | growth_hacker | A | 블로그 전환율 ... | 1단계 유입 | 리드 | - | 2.통념공격 | 단정 | 짧음 | 없음 | 6-5 | 단정 |
| 2 | growth_hacker | B | ... | 1단계 유입 | 체이닝 | 클리프행어 | 4.케이스 | 장면 | 중간 | 1-3개 | 6-6 | 여운 |
| 3 | strategist | A | ... | 2단계 신뢰 | 리드 | - | 1.내부자 | 질문 | 중간 | 없음 | 6-9 | 질문 |
| 4 | strategist | B | ... | 2단계 신뢰 | 체이닝 | 1/N 번호 | 6.프레임워크 | 단정 | 길음 | 4-6개 | 6-8 | 예고 |
| 5 | field_expert | A | [업종]... | 1단계 유입 | 리드 | - | 3.시행착오 | 역순 | 중간 | 없음 | 6-2 | 단정 |
| 6 | field_expert | B | ... | 3단계 전환 | 체이닝 | 역질문 | 7.손해강조 | 대화체 | 짧음 | 1-3개 | 6-7 | 여운 |
| 7 | agency_brand | A | [업종]... | 2단계 신뢰 | 리드 | - | 4.케이스 | 대화체 | 중간 | 없음 | 6-6 | 단정 |
| 8 | agency_brand | B | ... | 2단계 신뢰 | 리드 | - | 5.안티체크 | 두줄 | 짧음 | 없음 | 6-1 | 여운 |
```

### 표 다음에 검증 결과 자동 출력

```
[검증]
- 훅 패턴 다양성: 8개 글에 _가지 등장 (목표 ≥6)
- 내용 흐름 다양성: 8개 글에 _가지 등장 (목표 ≥4)
- 본문 길이 분배: 짧음 _, 중간 _, 길음 _ (목표 균형)
- 구조 분배: 리드 _, 체이닝 _ (체이닝 ≥3)
- 마무리 형식 분배: 질문 _, 단정 _, 여운 _, 예고 _ (예고 ≤1, 질문 연속 없음)
- 인간 티 글 포함: 예 / 아니오 (목표 ≥1)
- 어제와 겹치지 않게 변주: 예 / 아니오
- field_expert 트랙 A 업종: ___ (직전 3일과 겹치지 않음 확인)
- 토픽 태그: 페르소나별 2개 글 서로 다른 태그 사용 확인
```

모든 검증 항목이 통과되어야 분배표 확정. 하나라도 실패하면 표를 재조정해서 다시 출력.

---

## 분배표 확정 후

분배표가 확정되면, 사용자는 각 페르소나 프롬프트 파일을 첨부하면서 한 글씩 생성한다.
각 글 생성 시 사용자가 분배표의 해당 행을 페르소나 프롬프트의 "오늘 설정"에 그대로 옮겨 채운다.
